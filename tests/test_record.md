## 四、测试发现的问题与修复记录

### 问题1: Python 2/3 兼容性问题
- **现象**: 默认 `python` 命令调用 Python 2.7，不支持中文编码
- **修复**: 明确使用 Python 3.10 运行: `C:\Users\Administrator\AppData\Local\Programs\Python\Python310\python.exe`
- **状态**: 已修复 ✅

### 问题2: 时段分配导致大景点无法安排
- **现象**: 都江堰(4h)和青城山(5h)因超过单时段3h限制被跳过，Day 3/4 无景点
- **修复**: 放宽单时段上限至5h（上午/下午），使大景点可被正常安排
- **状态**: 已修复 ✅

### 问题3: all_day 景点时段限制
- **现象**: all_day 属性景点在固定时段景点之后安排时，可能因时段已满被跳过
- **修复**: 实现两轮分配策略——先安排固定时段景点，再安排 all_day 景点
- **状态**: 已修复 ✅

### 问题4: weather.py 缺少 List 类型导入
- **现象**: `NameError: name 'List' is not defined`
- **修复**: 在 typing 导入中添加 `List`
- **状态**: 已修复 ✅

### 问题5: 景点重复使用
- **现象**: 多日行程中同一景点可能被重复安排
- **修复**: 引入全局已使用景点集合 `global_used_names`，避免重复
- **状态**: 已修复 ✅
