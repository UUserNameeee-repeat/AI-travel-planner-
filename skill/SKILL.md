# AI 旅行规划师 (AI Travel Planner)

## 概述

AI旅行规划师是一个智能行程规划技能。它解决了传统旅行规划中信息分散、
决策复杂、耗时长的问题。用户只需提供基本需求（目的地、时间、预算、偏好），
即可获得一份包含每日行程、景点推荐、餐饮建议、交通方案、预算明细的完整规划。

## AI 核心价值

本技能的 AI 核心价值体现在：

1. **智能行程编排** —— 不是简单的景点列表，而是根据景点位置、开放时间、
   游玩时长、用户体力曲线进行智能排布，避免走回头路
2. **个性化推荐** —— 基于用户兴趣标签（文化/自然/美食/购物/冒险等）
   进行偏好匹配，推荐最合适的景点和活动
3. **预算智能分配** —— 根据目的地消费水平和用户预算自动分配
   住宿/餐饮/交通/门票/购物的预算比例
4. **上下文感知** —— 考虑天气、季节、节假日等因素调整行程建议

## 使用方式

### 基本用法

向 AI 助手描述你的旅行需求，例如：

> 我想国庆节去成都玩4天，预算5000元，喜欢美食和自然风光，
> 两个人出行，住中等酒店。

AI 助手会调用本技能，生成完整行程规划。

### 输入参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| destination | string | 是 | 目的地城市 |
| days | int | 是 | 出行天数 |
| budget | float | 否 | 总预算（元），默认不限 |
| travelers | int | 否 | 出行人数，默认1 |
| preferences | list | 否 | 兴趣偏好列表 |
| accommodation_level | string | 否 | 住宿等级：budget/standard/premium |
| start_date | string | 否 | 出发日期 YYYY-MM-DD |
| transportation | string | 否 | 偏好交通方式 |

### 兴趣偏好标签

- `culture` — 历史文化（博物馆、古迹、宗教场所）
- `nature` — 自然风光（山川、湖泊、公园）
- `food` — 美食探索（当地特色、街头小吃、餐厅）
- `shopping` — 购物（商场、市集、特色店）
- `adventure` — 冒险户外（徒步、攀岩、水上运动）
- `nightlife` — 夜生活（酒吧、夜市、演出）
- `photography` — 摄影打卡（地标、网红点）
- `relaxation` — 休闲放松（温泉、SPA、咖啡馆）

## 输出格式

技能输出一份结构化的旅行行程，包含：

1. **行程概览** — 目的地、天数、预算总览
2. **每日行程** — 上午/下午/晚上的活动安排
3. **景点详情** — 推荐理由、游玩时长、门票、提示
4. **餐饮推荐** — 早餐/午餐/晚餐建议
5. **交通方案** — 市内交通建议
6. **预算明细** — 各项费用分解
7. **实用提示** — 天气、注意事项、必备物品

## 工作流程

```
用户输入需求
    │
    ▼
┌─────────────────┐
│  解析需求参数    │
└────────┬────────┘
         │
    ▼
┌─────────────────┐
│  匹配目的地知识  │  ← references/destinations.json
└────────┬────────┘
         │
    ▼
┌─────────────────┐
│  生成候选景点池  │  ← 按偏好筛选
└────────┬────────┘
         │
    ▼
┌─────────────────┐
│  智能行程编排    │  ← scripts/planner.py
│  (路线优化)      │
└────────┬────────┘
         │
    ▼
┌─────────────────┐
│  预算分配计算    │  ← scripts/budget.py
└────────┬────────┘
         │
    ▼
┌─────────────────┐
│  生成完整行程    │  ← scripts/formatter.py
└────────┬────────┘
         │
    ▼
    输出行程方案
```

## 依赖文件

- `scripts/planner.py` — 核心行程规划引擎
- `scripts/budget.py` — 预算计算与分配
- `scripts/formatter.py` — 输出格式化
- `scripts/weather.py` — 天气查询模块
- `references/destinations.json` — 目的地知识库
- `references/budget_templates.json` — 预算模板
- `references/preferences.json` — 偏好标签映射
